|이름|담당 기능|
|---------|----------|
|**희라**|헬스장 소개 페이지, 지점 찾기, 지점 상세조회, 회원전용 페이지-수강신청, 이용권 구매|

|**경진**|Main페이지, CEO 정적페이지, 관리자 권한 부여|
|**산하**|공지사항 게시판, qna(1:1문의와 자주묻는질문) 게시판, 헬스장 후기 게시판|
